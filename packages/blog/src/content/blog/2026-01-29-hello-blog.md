## 안녕하세요

이것이 블로그의 첫 글입니다.

### 블로그의 특징

이 블로그는 다음과 같은 특징을 가지고 있습니다:

- **Astro 기반**: 빠른 정적 사이트 생성기 기반
- **마크다운 지원**: 마크다운으로 쉽게 글 작성
- **GitHub Pages 배포**: 자동 배포
- **다크모드 지원**: 시스템 설정에 따른 자동 테마 전환

### 기술 스택

- Astro 4.x
- TypeScript
- CSS

### 글 작성 방법

`src/content/blog/` 디렉토리에 마크다운 파일을 추가하면 자동으로 블로그에 반영됩니다.

파일명 형식: `YYYY-MM-DD-제목.md`

### Frontmatter 설정

각 글의 최상단에 다음과 같이 메타데이터를 설정합니다:

```markdown
---
title: "글 제목"
description: "글 설명"
pubDate: 2026-01-29
tags: ["tag1", "tag2"]
author: "작성자"
relatedDocs: ["https://link-to-docs"]
---
```

감사합니다!
